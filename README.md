# Arduino for Beginners — A Free, Open Curriculum

A free, open-source Arduino curriculum anyone can use to teach and learn hands-on electronics. Built to be picked up and run by any teacher, parent, librarian, or club, anywhere. It works with or without physical parts: every lesson can be done with a real Arduino kit or fully virtually in TinkerCAD, so cost and equipment are never a barrier.

## Proof it works

I built this curriculum and ran it as a live program at my local library. Registration overflowed within 48 hours of posting. The session was capped at 12 people for space, but 15 signed up and all 15 showed up, all within the first two days. My team and I surveyed attendees afterward, and most reported feeling more comfortable with electronics and enjoying it more than before the session.

## What's inside

Eight fully structured 90-minute lessons, taught in person or virtually in TinkerCAD, progressing from a single blinking LED through circuits, buttons, analog input, servos, and core programming concepts like loops and switch statements. The set includes two open-ended capstone projects where students design and build their own circuits, plus working code examples for every non-capstone lesson. Each lesson comes with a full lesson plan, safety notes, common-issues troubleshooting, and teacher guidance.

## What you need to run it

Either a hardware kit per student (Arduino Uno, breadboard, LEDs, resistors, button, potentiometer, servo, and jumper wires), or just a computer with internet for the free TinkerCAD virtual track. No prior experience with electronics is needed, either to learn from these lessons or to teach them.

## How to use / adopt it

Clone or download this repository. Lessons are numbered and meant to be taught in order, since each builds on the last. Start with Lesson 1, decide whether your group is using physical kits or TinkerCAD, and follow the lesson plan, which includes setup steps, the build, the code, and troubleshooting. If you adopt this for your own group, or want to contribute, translate, or adapt it, open an issue or reach out. I would love to see it used and improved.

## Why it exists

I taught myself electronics, and I would have loved a free, structured on-ramp like this when I started. I made it so the next beginner, anywhere, has the resource I wished I'd had.
